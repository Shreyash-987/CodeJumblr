How it Works

Welcome to the Substitution Cipher tool! This tool allows you to encrypt or decrypt messages using a custom substitution key. The Subsitution Cipher is a simple yet effective encryption method that takes each letter input and replaces it with a corresponding letter from a fixed key. By either randomly generating or importing a unique 26-letter key, you can easily encrypt and decrypt messages, making it a great tool for secure communication.
